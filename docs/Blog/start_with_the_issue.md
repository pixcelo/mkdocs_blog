# イシューとは

`issue`の意味は

`an important topic or problem for debate or discussion`

「重要な議題、あるいは問題」<br>

物事に取り組む際に、それは「イシューであるか」を問う必要がある

## イシューから始めよ

`Think ahead of the problem`

「先んじて考えよ」

全ての物事に均等に取り組むことでは、生産性を上げることはできない<br>

* 優先順位を設けることが必要となる

「イシューであるか」を見極め、一番重要なイシュー（解決することで価値を生む問題）にリソースを振ることが重要
