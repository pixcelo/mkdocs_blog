# 商用利用が可能なOSSのライセンス

* ライブラリ選定の際に注意したいライセンス規約
* 商用利用なものだけをまとめた
* 詳細については原文を必ず確認すること

## MIT

MITライセンス `MIT License`

プログラムを改変したり、自作のソースに組み込んだ場合は、OSSとしてソースコードを公開しなくても良い

* 制限がゆるい
* 再頒布時には著作権とライセンスを表示
* 商用利用が可能

## Apache2

Apache ライセンス `Apache License`

* オリジナルから修正点がある場合には明示する
* 再頒布時には著作権とライセンスを表示
* 利用者に特許使用権を付与
* 商用利用が可能
