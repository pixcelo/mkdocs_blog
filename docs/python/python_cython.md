# Python Cythonによる高速化

Cythonは、PythonにC言語の型が導入されたプログラミング言語

メリット
- 数倍〜数百倍の高速化を実現できる
- 変数の型宣言ができる

デメリット
- 学習コスト(日本語のドキュメントが少ない等も含む)
- `try except`など使えない機能がある
