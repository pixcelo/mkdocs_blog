# 工事中

## HTTP 動詞

| HTTPメソッド | 説明                                       |
|--------------|--------------------------------------------|
| GET          | データを取得する                            |
| POST         | リソースを作成するデータを送信する          |
| PUT          | リソースを更新するデータを送信する          |
| DELETE       | リソースを削除する                          |
