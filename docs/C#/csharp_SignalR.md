# C# SignalRによるリアルタイム通信

## Topic 

`SignalR`を使ってリアルタイム通信を行う

## OverView

`SignalR`とは、リアルタイムWeb機能を提供するライブラリ

- 複数のプラットフォームに対応
- サーバーとクライアント間のトランスポート方法を自動的に最適化
- OSS

## Usage

visual studio でASP.NET Core Webアプリケーションのプロジェクト作成

ライブラリ マネージャー (LibMan)
