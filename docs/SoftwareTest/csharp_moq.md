# Moq

Moqは、単体テスト用モックフレームワーク<br>
テスト用の代替オブジェクトを作成するための.NETライブラリ<br>
ネットワーク通信、データベースアクセスなどの外部に依存する機能を代替するオブジェクトをMoqで作成できる

## Usage

nugetからインストール
```
dotnet add package Moq --version 4.18.2
```
